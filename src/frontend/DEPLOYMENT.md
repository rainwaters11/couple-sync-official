# Couple Sync - Deployment Guide

## Overview
Couple Sync is a couples relationship habit app that runs on the Internet Computer. It requires no login and uses 6-digit room codes for optional cross-device syncing.

## Build Instructions

1. Install dependencies:
   ```bash
   cd frontend
   pnpm install
   ```

2. Build the frontend:
   ```bash
   pnpm build
   ```

3. The built files will be in `frontend/dist/`
